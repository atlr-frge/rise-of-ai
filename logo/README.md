# Materiality, Logo

The brand is **Materiality**; the logo is the **wordmark**. Primary treatment: `MATERIAL` in
off-white, `ITY` in ember. Confirmed 2026-06-02.

## Files
| File | Use |
|---|---|
| `materiality-wordmark.svg` / `.png` | **Primary** two-tone wordmark (transparent). The logo. |
| `materiality-lockup.svg` / `.png` | Wordmark + "An Atelier Superintelligence briefing" descriptor. |
| `materiality-wordmark-white.svg` / `.png` | Single-color (mono) wordmark; recolor fill for all-white or all-ember. |

## Specs
- **Type:** Inter, weight 900, letter-spacing ≈ -3 (tight). Swap to a licensed display face later
  only if we want more distinctiveness; Inter Black is the working standard.
- **Colors:** off-white `#FAFAFA`, ember `#E85D04`, near-black `#0A0A0A` ground.
- **Clearspace:** keep margin ≥ the cap-height of the "M" on all sides.
- **Min size:** wordmark legible to ~22px tall; below that use the mono version.
- **On ember:** use the all-white mono wordmark (reversed).

## Do / Don't
- Don't recolor `ITY` to anything but ember; don't split the two-tone differently.
- Don't add effects, outlines, or shadows. Flat only.
- Don't stretch or re-track. Use the provided files.
